What is a Computer?

A computer is an electronic device that processes data and performs tasks according to instructions called programs. Computers are used for work, education, entertainment, communication, and much more.

Main Parts of a Computer

CPU (Central Processing Unit): The CPU is the brain of the computer. It carries out instructions and performs calculations.

Memory (RAM): RAM temporarily stores data that the computer is using right now so it can access it quickly.

Storage (SSD/HDD): Storage keeps your files, programs, and operating system even when the computer is turned off.

Motherboard: The motherboard is the main circuit board that connects all the parts of the computer together.

Graphics Card (GPU): The GPU handles images, videos, and graphics-heavy tasks, especially for gaming and design work.
